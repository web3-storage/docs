# How to generate an API token

In this how-to guide, **you'll learn how to generate a Web3.Storage API token** so that you can interact with the service programmatically through the [JavaScript client library](../reference/client-library.md) or using the command line.

You'll need a free Web3.Storage account in order to generate an API token. If you already have an account, read on. If not, have a look at the [quickstart guide](../README.md#quickstart) to get up and running in just a few minutes.

## Create a new token


1. Click **Profile** to go to your [profile page](web3.storage/profile).
1. Scroll down to **API tokens** and click **Create an API token**.
1. Enter a descriptive name for your API token.

    :::tip
    Developers usually assign individual API tokens for different applications. Doing so makes it easier to revoke access in the future to specific apps or websites.
    :::

1. Click **Create**.
1. The website will redirect you to `/tokens`, which shows a [list of all your API tokens](web3.storage/tokens). You can copy and delete your API tokens from this page.
